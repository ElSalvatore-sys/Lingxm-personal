## Profile-Specific Vocabulary Strategy

Each profile has a **personalized vocabulary file** (data/{profile}-vocab.json) containing:
- Words tailored to proficiency level
- Specialization-specific terminology
- Cross-language vocabulary connections
- Progressive difficulty increase

**Example Structure:**
```json
{
  "profile": "kafel",
  "totalWords": 540,
  "dailyTarget": 30,
  "languages": {
    "de": {
      "level": "B2-C1",
      "specialization": "IT",
      "wordCount": 360
    },
    "en": {
      "level": "C1-C2",
      "wordCount": 180

  "words": [

      "word": "debuggen",
```

## Data Privacy & Security

### Profile Data Storage
- **Location:** Local device only (IndexedDB + localStorage)
- **Encryption:** PIN-protected profiles
- **Backup:** Manual export/import (JSON format)
- **Privacy:** No cloud sync, no external tracking

### Profile Switching
- **Authentication:** 4-digit PIN per profile
- **Session Management:** Auto-logout after inactivity
- **Data Isolation:** Complete separation between profiles
- **Progress Preservation:** Resume feature saves position across app restarts
